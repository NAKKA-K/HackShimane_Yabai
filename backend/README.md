# やばい

# Install
```
    $ gem install bundler
    $ bundle install --path vendor/bundler
```

# Server exec
```
    $ bundle exec rails s
```

# DB exec
```
    $ bundle exec rails db:migrate
```

# DB reset
```
    $ bundle exec rails db:reset
```

# DB test data
```
    $ bundle exec rails db:seed
```
